================================================================================
                           SAMPLE DATA STRUCTURE
================================================================================

Each extracted line item contains the following fields:

  • order_id           - Links back to original Trello card
  • line_index         - Position within the order (1, 2, 3...)
  • location_group     - Section header if present (e.g., "STORE FRONT")
  • order_class        - "Supply", "Install", or "Supply & Install"
  • quantity           - Number of items (e.g., 2, 50, 500)
  • price              - Dollar amount
  • raw_price_text     - Original price text for verification
  • price_validated    - TRUE/FALSE validation flag
  • width_in           - Width in inches
  • height_in          - Height in inches
  • raw_dimensions_text- Original dimension text (e.g., "24x36")
  • material           - Product material (e.g., "Aluminum Composite Panel")
  • description        - Item description
  • colour             - Color if specified
  • extra_notes        - Additional details
  • raw_line_text      - Original source text
  • llm_confidence     - Extraction confidence (high/medium/low)
